# AI Museum Guide

Point your camera at paintings and let AI explain the artwork with voice narration and quiz mode.

## Features

- Camera capture for paintings
- AI-powered artwork analysis using OpenRouter/Gemini
- Voice narration with Web Speech API
- Quiz mode for students
- Progress tracking and scoring

## Tech Stack

- React + Vite
- OpenRouter API (Gemini 2.0 Flash)
- Web Speech API for text-to-speech

## Getting Started

```bash
npm install
npm run dev
```

Open http://localhost:3000 and enter your OpenRouter API key in Settings.
